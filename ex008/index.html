Principais formatações

NEGRITO (ou destaque)

Texte em NEGRITO utilizando a tag B (não semântica).

Texto em NEGRITO utilizando a tag STRONG (semântica).

Itálico (ou ênfase)

Texto em itálico com a tag I (não semântico).

Texto em itálico com a tag EM (semântico).

Texto marcado

Podemos também criar um texto marcado, como um grifa texto.

Texto grande e pequeno

Estamos criando um texto grande e um pequeno neste parágrafo.

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado.

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase de que ele foi adicionado depois.

Existe também o texto sublinhado, utilizando a tag U.

Texto sobrescrito

Serve para inserir coisas do tipo x30+10

Texto subescrito

Para inserir coisas do tipo H20.
